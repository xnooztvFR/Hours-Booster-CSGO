# Hours Booster CSGO


# Installation

1) Télécharger NodeJS si vous ne l'avez pas, lien: https://nodejs.org/dist/v6.10.0/node-v6.10.0-x64.msi
2) Lancer "Install.cmd"

Bon boost d'heures !

# Configuration & Lancement du bot

1) Ouvrir "IdlerConfig.js" avec notepad++ (recommandé), bloc note ou un autre éditeur.
2) Sur la ligne:  config.username = "MONCOMPTESTEAM";
				  Modifier "MONCOMPTESTEAM" par votre nom d'utilisateur steam
   Sur la ligne:  config.password = 'MONMDP';
				  Modifier "MONMDP" par le mot de passe du compte steam (exemple: "azerty123")

3) Lancer "StartIdle.cmd", rentrer le code steam guard si demandé,
   vous devriez voir votre compte entrain de jouer à CSGO ou autre jeu.
